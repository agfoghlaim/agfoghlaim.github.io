Synth App

This app is a synthesyer that records sound and plays back music and noise. It's mostly based on the Web Audio API and oscillators. The main page has a one octive synthesyer keyboard where the user can play basic melodys. The keyboard also has 4 drum sounds and distortion on/off buttons. These vibrate the phone with the beat. There are record, stop record and playback buttons on this page also. On the right menu off this view there are options to change the synthesyer sound – sine, triangle, square and sawtooth waves. To the left of the main view there is an “experiment” view. This is for making noise. A range slider controls the frequency of the sound while radio buttons allow the user to select the type of wave used. This choice is stored in local storage. The second range slider controls the volume. The Chord mixer tab has an initialise oscillator button, a start and stop oscillator as well as volume control so the user can create cords to play melodies over using the keyboard on the main page.
